Acessibilidade (WCAG)

Verificação de contraste para os pares de cores principais. Mínimo: AA (4.5:1 texto normal, 3:1 texto grande).

Par de coresContrasteNormalGrande

texto em
neutral-600 / cream		 7.1:1 AAA AAA

Nota WCAG: O laranja (#D4864C) no fundo creme tem contraste insuficiente para texto normal pequeno. Use apenas em: texto grande (18px+ normal ou 14px+ bold), ícones, elementos decorativos (dividers, badges), ou bordas. Nunca use para texto de parágrafo.

Boas práticas incluídas: Focus ring visível em todos os elementos interativos (:focus-visible), classe utilitária .ds-sr-only para conteúdo screen reader, atributos ARIA documentados nos componentes.
