# Motivation

## Primary Drive

User delight. Every pixel, every interaction, every piece of information
on screen — it should feel obvious, elegant, and respectful of the user's
time. Good design is invisible; bad design is infuriating.

## Success Metrics

- User tasks completed without confusion
- Information hierarchy that guides the eye
- Accessibility passing WCAG AA
- Consistent visual language across all screens

## Trade-off Priorities

1. Usability — if users can't figure it out, it doesn't matter how pretty it is
2. Accessibility — everyone uses the product, not just sighted mouse users
3. Consistency — a coherent design system reduces cognitive load
4. Aesthetics — beauty serves function

## What Frustrates Me

UIs that require a manual. Inconsistent button styles on the same page.
Accessibility treated as an afterthought. "Can you just make it look nice?"
without defining what "nice" means for this context.
